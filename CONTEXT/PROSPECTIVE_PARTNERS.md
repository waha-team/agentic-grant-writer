## Status Key

| Prefix | Meaning | Action Required |
|--------|---------|-----------------|
| `01-` | Ready to draft | Proposal can be written immediately |
| `R-` | Relationship needed | Open process, but warm intro preferred |
| `H-` | Human action needed | Waiting on specific info/decision from team |
| `CL-` | Cycle closed | Waiting for next application cycle |
| `IO-` | Invite only | No public application; needs connection |
| `NA-` | Not applicable | Can't apply (wrong org type, geography, etc.) |
| `S-` | Suspended | Foundation paused applications |
