# Stage 00: Projects

> This folder holds project briefs — the initiatives that need grant funding. Each project gets its own subfolder.

---

## How This Stage Works

Humans create project briefs here. When ready, move the project folder to `01-prospect-research/`.

## For Humans

1. Copy `_TEMPLATE/` and rename (e.g., `waha-farsi-localization/`)
2. Fill in `PROJECT_BRIEF.md`
3. Move the folder to `01-prospect-research/`

## For Claude Code

If asked to help draft a brief: read `CONTEXT/OUR_ORGANIZATION.md`, ask the human what initiative needs funding, generate a draft using the template structure.

## What a Good Project Brief Contains

At minimum:
- A clear project name and one-sentence description
- Who it serves (specific population, geography, language)
- What will happen (activities and deliverables)
- How much it costs (budget range)
- What success looks like (measurable outcomes)
- Timeline

The more specific and quantified the brief is, the better the prospect research and proposals will be.
