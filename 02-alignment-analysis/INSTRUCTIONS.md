# Stage 02: Alignment Analysis

> **Your job:** Visit each prospect's website, evaluate fit, and produce a scored shortlist of the top 8–12 foundations to pursue.

---

## Before You Start

1. Read `CONTEXT/OUR_ORGANIZATION.md`
2. Read `CONTEXT/SEARCH_CRITERIA.md`
3. Read the `PROJECT_BRIEF.md` inside each project subfolder
4. Read the `PROSPECTS.md` that was created in Stage 01

## What to Do

For each project subfolder found in this directory:

### Step 1: Deep-Dive Each Prospect
For every foundation listed in PROSPECTS.md:

1. **Visit the foundation's website.** Read their mission, program areas, grant guidelines, and any available lists of past grantees.
2. **Check for specific alignment signals:**
   - Do they explicitly fund missions, evangelism, or disciple-making?
   - Do they fund Bible translation, Scripture access, or literacy?
   - Do they fund technology projects or innovation in ministry?
   - Do they fund work in the same geography as this project?
   - Have they funded organizations similar to Waha in size and mission?
   - Do their values overlap with Waha's? (see CONTEXT/SEARCH_CRITERIA.md "Values Alignment Signals")
3. **Check for disqualifiers:**
   - Application is invitation-only with no warm path
   - Minimum grant size is larger than what we're requesting
   - They haven't made grants in 2+ years (dormant foundation)
   - They explicitly exclude international work or religious organizations
   - Their geographic or thematic focus doesn't actually match

### Step 2: Score Each Prospect
Use this rubric (score 1-10 for each dimension, then average):

| Dimension | Score | What to Assess |
|-----------|-------|----------------|
| **Mission Alignment** | 1-10 | How closely does their stated mission match this project? |
| **Track Record** | 1-10 | Have they funded similar work before? (Check grantee lists, 990s) |
| **Size Match** | 1-10 | Is their typical grant size in our target range? |
| **Accessibility** | 1-10 | Can we actually apply? (Open apps > LOI > invitation-only) |
| **Geographic Fit** | 1-10 | Do they fund work in the regions this project serves? |
| **Relationship Potential** | 1-10 | Is there a path to building a relationship? (Public contact info, events, LinkedIn presence) |

**Overall Score** = Average of all dimensions

### Step 3: Produce the Shortlist
Create a file called `ALIGNMENT_ANALYSIS.md` inside the project's subfolder:

```
# Alignment Analysis for [Project Name]

*Generated: [DATE]*
*Prospects analyzed: [NUMBER]*
*Shortlisted: [NUMBER]*

---

## Recommended: Apply to These (Score 7+)

### [Foundation Name] — Score: X.X/10
| Dimension | Score | Notes |
|-----------|-------|-------|
| Mission Alignment | X | [brief note] |
| Track Record | X | [brief note] |
| Size Match | X | [brief note] |
| Accessibility | X | [brief note] |
| Geographic Fit | X | [brief note] |
| Relationship Potential | X | [brief note] |

**Recommended framing for Waha:** [2-3 sentences on how to position Waha for this specific funder]
**Estimated grant ask:** $[X]
**Application deadline/cycle:** [info]
**Next step:** [what needs to happen — e.g., "submit LOI by March 15" or "open application, apply anytime"]

[Repeat for each recommended foundation]

## Worth Watching (Score 5-6.9)

[Same format but briefer — these go in the "maybe later" pile]

## Not Pursuing (Score <5)

[List with one-line reason for each: "Mission doesn't align", "Invitation only, no warm path", etc.]
```

### Step 4: Move Forward
After creating ALIGNMENT_ANALYSIS.md, move the project subfolder to `03-contact-research/`.

## Decision Points

**Default to momentum.** Keep moving unless:

- Fewer than 4 foundations score 7+ → Add to HUMAN_TODO.md: "Only [X] strong prospects for [project]. Options: (a) proceed with what we have, (b) go back to 01 for more prospect research, (c) reframe the project. Recommendation: [your recommendation]."
- A foundation looks like a perfect fit but is invitation-only → Add to HUMAN_TODO.md: "[Foundation] scored 9+/10 for [project] but is invitation-only. Worth pursuing a warm introduction. Check if anyone on the Waha team/board has a connection."
- You discover a foundation has a specific deadline coming up soon (<30 days) → Add to HUMAN_TODO.md: "URGENT: [Foundation] deadline is [DATE] for [project]. Need to fast-track through remaining stages."

## Quality Standards

- Every recommended foundation must have been verified on their actual website (not just from a database listing)
- Scores must be justified with brief notes, not just numbers
- The "recommended framing" section is critical — it tells Stage 05 how to write the proposal
- Be honest about weak scores. A 6 is fine if the reasoning is sound.
